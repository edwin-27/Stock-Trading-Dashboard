# Stock Trading Dashboard

## Overview

Stock Trading Dashboard is a responsive web application user interface designed for investors to monitor market activity, manage portfolios, and execute stock trades. The project focuses on modern dashboard design principles and responsive layouts for both desktop and mobile devices.

## Features

- Responsive dashboard layout
- Sidebar navigation menu
- Portfolio overview cards
- Stock watchlist
- Market statistics
- Trading form (Buy/Sell stocks)
- Transaction history table
- Search functionality
- User profile section
- Mobile-friendly design

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5

## Pages

### Dashboard
Provides an overview of portfolio performance, market activity, and recent transactions.

### Portfolio
Displays owned stocks and investment values.

### Trade
Allows users to simulate buying and selling stocks through interactive forms.

### Market
Shows market trends, top gainers, top losers, and popular stocks.

### Settings
Contains account and notification preferences.
